# kotlin-lodash

kotlin-lodash is a Kotlin library that provides a set of utility functions for common programming tasks. This library aims to simplify everyday operations on strings, collections, and arrays, making your Kotlin development experience more efficient and enjoyable.

## Installation

To include kotlin-lodash in your project, add the following dependency to your `build.gradle.kts` file:

```kotlin
dependencies {
    implementation("lodash.kotlin:kotlin-lodash:0.0.1)
}
```

## Usage

```kotlin
import lodash.trim

val trimmedString = trim("  Hello, World!  ")
println(trimmedString) // Output: "Hello, World!"
```
